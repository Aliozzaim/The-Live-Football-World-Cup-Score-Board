# Live Football World Cup Scoreboard

## Assumptions

- Team names are **case-sensitive** (e.g., `"Brazil"` ≠ `"BRAZIL"`).
- Scores must be **non-negative integers** (0 or greater). Negative or non-integer scores are invalid.
- A match cannot exist twice between the same **home** and **away** teams simultaneously.
- Unlimited matches can be added; users are not restricted by any maximum limit.

## Installation

```bash
npm install
```

## Running Tests

```bash
npm test
```
